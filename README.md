# Análisis, diseño e implementación de Jorgestor

## Contexto

En estas dos asignaturas de Ingeniería del Software (I y II) nos hemos centrado en la metodología **RUP**, esta metodología se divide en disciplinas (requisitos, análisis, diseño, implementación, pruebas y transición)

Durante el primer cuatrimestre vimos la **disciplina de requisitos** y la pusimos en práctica en un proyecto en grupos en el que cada grupo tenía un cliente ficticio. En este proyecto tuvimos que desarrollar todas las actividades propias de esta disciplina. Este proyecto en el caso de mi grupo es **[Jorgestor](https://github.com/martinlopez7/25-26-IdSw1-SdR)**

Para este segundo cuatrimestre se han tenido que aplicar las siguientes disciplinas de RUP, es decir, **análisis**, **diseño** e **implementación** en el proyecto anteriormente mencionado, pero con una condición: hacerlo todo con IA.
